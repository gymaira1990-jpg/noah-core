# 项目6文件模板

每个诺亚核心项目包含6个标准文件。进行中阶段维护4个动态文件，归档时补全2个文件。

## 动态文件（进行中实时更新）

### 1. `_进度.md`
```markdown
# [项目名称] — 进度追踪

## 当前状态
阶段：进行中
最后更新：YYYY-MM-DD

## 已完成
- [ ] 任务一
- [x] 任务二

## 待完成
- [ ] 任务三
```

### 2. `_对接全记录.md`
```markdown
# [项目名称] — 对接全记录

> 时间跨度：YYYY-MM-DD
> 参与方：GCat ↔ 执行Agent

## YYYY-MM-DD
**[GCat → Agent]**
"需求描述"

**Agent做了什么：**
- 操作记录
```

### 3. `_经验技能包.md`
```markdown
# [项目名称] — 经验技能包

记录踩坑经验和可复用知识。

## 架构经验
### [问题描述]
- 现象：
- 原因：
- 解决：
```

### 4. `_外部资源索引.md`
```markdown
# [项目名称] — 外部资源索引

## 核心路径
| 资源 | 路径 | 说明 |

## 相关技能
| 技能名称 | 说明 |

## 相关项目
| 项目 | 路径 | 关联 |
```

### 5. `_标签索引.md`
```markdown
# [项目名称] — 标签索引

## 核心标签
- tag1
- tag2

## 技术标签
- tech1

## 状态
- 进行中
```

## 归档文件（仅项目完成时生成）

### 6. `_项目描述.md`
项目概述、架构、功能说明、技术参数。

### 7. `_里程碑.md`
```markdown
# [项目名称] — 里程碑

> 验收时间：YYYY-MM-DD
> 验收人：GCat
> 状态：✅ GCat确认完成
```
