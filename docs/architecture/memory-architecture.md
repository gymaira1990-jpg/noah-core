# 诺亚核心 · 记忆架构

> 双层记忆存储 + 温度分层生命周期

## 设计原则

### 省Token
- memory 工具只存索引指针（~2200字符限额）
- 全量文档存 vault/ 永久仓库
- 高频技能自动提升到公共区域

### 记忆分类
| 类型 | 内容 | 生命周期 |
|------|------|---------|
| 永久留存 | 姓名、灵魂、核心特质、精简履历 | 永不删除（→先贤祠） |
| 动态操作 | 软日志、会话记录 | 归档时清理 |
| 能力记忆 | 技能包、经验教训 | 按热度升级流转 |

### 三卡一体
每个AI角色三类记忆：
1. **身份卡** → vault/team/[角色名]/
2. **软日志** → 操作留痕，归档即删
3. **技能包** → 个人→公共→永久，按热度流转

## 温度迁移参数

| 迁移 | 阈值 | 动作 |
|------|------|------|
| 热→普通 | 7天无访问 | 物理移文件到普通记忆目录 |
| 普通→冷 | 30天无访问 | 压缩为摘要+关键参数 |
| 冷→仓库 | 90天无访问 | 一句话+标签+路径指针 |
| 任意→热 | 被访问 | 恢复完整内容 |

## 文件结构

```
诺亚核心/项目/
├── _温度索引.md        ← 全项目温度追踪
├── _记忆执行规则.md     ← 铁律+生命周期
├── 热记忆/             ← 活跃项目
├── 普通记忆/           ← 冷却中
├── 冷记忆/             ← 压缩摘要
├── 仓库记忆/           ← 一句话指针
└── [项目名称]/         ← 进行中项目
    ├── _进度.md
    ├── _对接全记录.md
    ├── _经验技能包.md
    ├── _外部资源索引.md
    ├── _标签索引.md
    ├── _项目描述.md     ← 归档时才生成
    └── _里程碑.md       ← 归档时才生成
```
